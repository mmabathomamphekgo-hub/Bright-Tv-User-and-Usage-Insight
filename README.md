# Bright-Tv-User-and-Usage-Insight

 **Project Summary**
An end-to-end data analytics project analyzing user growth, engagement, and content consumption patterns on BrightTV. This dashboard translates raw usage data into actionable insights that inform content strategy, marketing effectiveness, and user retention.

**Business Problem:**
BrightTV needed to understand:
What drives user growth and engagement spikes
Whether new users are being retained effectively
Which content types and time slots maximize engagement
This project answers those questions through data-driven analysis.

 **Key Impact (What I Discovered):**
 DAU (Daily active users) grew steadily, but sharp spikes were campaign-driven and short-lived
 New user acquisition peaks (max: 520 users) directly drove engagement spikes
 Retention is improving, but many users from campaigns do not return
 Peak engagement occurs in Afternoon–Evening (56% of total watch time)
 Fridays drive the highest watch hours (1.2M)—due to higher per-user engagement, not more users
 Sports content dominates total engagement, while
 Entertainment content drives deeper per-user viewing
 Users exhibit long session durations (~1,300 mins avg), indicating strong engagement among active users
 
 **What This Means (Business Value):**
 Campaigns are effective for acquisition, but need stronger retention strategies
Sports content is a key growth driver → ideal for scaling engagement
Music/entertainment can be leveraged to increase user stickiness
Optimal engagement windows enable better:
Ad placement
Content scheduling
Push notifications

**Tools & Skills Demonstrated:**
SQL – Data extraction, cleaning, and transformation
Data Visualization (Power BI / Tableau) – Interactive dashboard design
Exploratory Data Analysis (EDA) – Trend and pattern identification
Business Analysis – Translating data into strategic recommendations
 
**Dashboard Highlights:**
User growth trends (DAU, new vs returning users)
Time-based engagement analysis (hourly & weekly patterns)
Content performance (top channels & categories)
Session behavior insights

**Key Takeaways for Stakeholders:**
 Invest in retention mechanisms post-campaign
Double down on sports content for scale
Use peak hours (12:00–20:00) for releases and promotions
Personalize experiences to convert one-time users into loyal viewers

**Next Steps:**
Cohort retention analysis
User segmentation (behavioral profiling)
Predictive analytics (churn & engagement forecasting)
A/B testing for content and notification strategies
